To use this program, Wireshark must be installed and set up correctly with the path C:\Program Files\Wireshark

This is a Windows plug-in for wireshark that allows easy IP pulling for many different filters.
Simply run the BAT file in the source folder (Python 3.8) or install the application using the provided installer.
Currently only works for IPv4.

Please use this program responsibly.  Enjoy :)
